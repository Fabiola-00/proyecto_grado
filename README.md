# SICOSE
## Descripción
SICOSE es un Sistema de Control y Seguimiento. El Objetivo es que se lleven registro del personal de Instructores del SBRAB, tanto de las asistencias, cursos, operaciones y de la evaluación que se les realiza.

## Instalación

Para poder instalar SICOSE solo se necesita contar con un servidor Web con Apache. No es necesario contar con MySQL ya que este Sistema utiliza una base de datos local (SQLite).
